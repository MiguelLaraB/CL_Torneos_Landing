## ADDED Requirements

### Requirement: Upcoming Tournaments Sidebar
The system MUST display a list of upcoming tournaments to the right of the Hero section on desktop (or stacked on mobile).

#### Scenario: User views upcoming tournaments list
- **WHEN** the user views the landing page
- **THEN** they see a heading "NEXT TOURNAMENTS" alongside a predefined list of tournaments

### Requirement: Tournament Entry formatting
Each tournament entry MUST display the game logo/name, date, and prize pool.

#### Scenario: User views a tournament entry
- **WHEN** the user inspects a tournament item in the list
- **THEN** it displays the game (e.g., League of Legends or Valorant), a date (e.g., "19 - Dic"), and a prize amount (e.g., "700$")
